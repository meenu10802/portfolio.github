Poetry Generator

Poetry Generator is a website that takes input keywords and generates a poem using those keywords with a suitable image. It is responsive a machine learning project using Python Flask and TensorFlow. The model is hosted on Hugging Face and is connected to the UI design using JavaScript. It also provides a library of poems where you can search for any poem; it has a vast collection of poems. Additionally, it provides a list of poets where you can search by the author's name.
